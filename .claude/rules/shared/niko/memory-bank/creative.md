# Creative

Creative documents are created during the creative phase of a task.
They are used to document the design decisions made for a given feature or capability.

## Files

Creative documents are stored in the `memory-bank/creative/` directory.
Creative documents are named in the format `creative-[feature_name].md`.

## Format

Different kinds of creative exploration will have different formats. Defer to the specific creative phase documentation for the format.
